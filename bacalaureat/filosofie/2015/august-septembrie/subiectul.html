Subiectul I

Scrieţi pe foaia de examen litera corespunzătoare răspunsului corect, pentru fiecare dintre situaţiile de mai jos. Este corectă o singură variantă de răspuns.

1. Din punct de vedere filosofic, împlinirea posibilităţilor noastre prin faptele pe care alegem să le facem desemnează:

  1. o formă de constrângere

  2. raportul adevăr-fals

  3. o formă de discriminare

  4. sensul existenţei

2. Etica, înţeleasă ca parte practică a moralei, vizează reflecţii privitoare la:

  1. constituirea statului modern

  2. fericirea ca scop al vieţii

  3. modul în care stabilim adevărul unui enunţ

  4. conceptul de filosofie

3. Eutanasia constituie o temă de studiu pentru etica:

  1. teleologică

  2. hedonistă

  3. eudaimonistă

  4. aplicată

4. Realizarea libertăţii negative, în practică, presupune:

  1. intervenţia sporită a statului asupra cetăţenilor

  2. garantarea drepturilor şi libertăţilor individuale

  3. prezenţa constrângerilor impuse de puterea politică

  4. asigurarea resurselor necesare pentru protecţie socială

5. Filosoful J. P. Sartre consideră că:

  1. omul este condamnat să fie constrâns

  2. fiinţa umană deţine o libertate limitată

  3. omul este condamnat să fie liber

  4. existenţa umană este marcată de iresponsabilitate

6. Sintagma Oamenii se nasc liberi vizează un drept:

  1. natural

  2. pozitiv

  3. economic

  4. neutru

7. În opinia filosofului Im. Kant, judecăţile sintetice exprimă o cunoaştere:

  1. independentă de experienţă

  2. extensivă

  3. explicativă

  4. analitică

8. Opinia mărul este roşu este adevărată dacă şi numai dacă prin confruntare cu realitatea se constată că mărul este roşu, potrivit teoriei adevărului:

  1. utilitate

  2. coerenţă

  3. corespondenţă

  4. pragmatic

9. Aristotel este unul dintre reprezentanţii eticii:

  1. hedoniste

  2. eudaimoniste

  3. deontologice

  4. aplicate

10. În contextul în care deosebirile dintre oameni sunt valorizate în societatea contemporană în moduri diferite, conceptul de discriminare se referă la tratamentul:

  1. egal al oamenilor dintr-o anumită comunitate

  2. diferenţiat al oamenilor cărora li se acordă o a doua şansă

  3. diferenţiat şi nedrept al unora dintre oameni pe baza unor caracteristici ale acestora

  4. egal al oamenilor în baza drepturilor omului.
