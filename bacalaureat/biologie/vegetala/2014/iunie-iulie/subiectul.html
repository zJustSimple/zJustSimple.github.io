Subiectul I

A. Scrieţi, pe foaia de examen, noţiunile cu care trebuie să completaţi spaţiile libere din afirmaţia următoare, astfel încât aceasta să fie corectă.
Pepsina este o enzimă din compoziţia sucului gastric, cu rol în digestia proteinelor.

B. Daţi două exemple de boli ale sistemului excretor la om; scrieţi în dreptul fiecărei boli câte un exemplu de manifestare a acesteia.

Pielonefrita este o afecțiune a sistemului excretor la om și se manifestă prin dureri lombare, febră, dureri și usurime la urinare.
Glomerulonefrita, indiferent de cauzele ei, se manifestă frecvent printr-un sindrom nefritic, caracterizat de hematurie, hipertensiune, edeme, insuficiență renală. 

C. Scrieţi, pe foaia de examen, litera corespunzătoare răspunsului corect. Este corectă o singură variantă de răspuns.

1. Boala Parkinson este o afecţiune a sistemului:

  1. digestiv
  2. excretor
  3. nervos central
  4. reproducător masculin


Răspuns corect: c

2. Organul alcătuit din numeroase unităţi microscopice numite nefroni este:

  1. encefalul
  2. inima
  3. ovarul
  4. rinichiul


Răspuns corect: d

3. Sunt cordate:

  1. amfibienii
  2. artropodele
  3. bacteriile
  4. viermii


Răspuns corect: a

4. Respiraţia aerobă:

  1. constă în oxidarea substanţelor organice
  2. este specifică mamiferelor
  3. produce o cantitate mică de energie
  4. se mai numeşte şi fermentaţie


Răspuns corect: a

5. Receptorii vizuali:

  1. fac sinapsă cu neuronii multipolari
  2. sunt absenţi în pata galbenă
  3. sunt celule chemosensibile
  4. sunt localizaţi în retină


Răspuns corect: d

D. Citiţi, cu atenţie, afirmaţiile următoare. Dacă apreciaţi că afirmaţia este adevărată, scrieţi, pe foaia de examen, în dreptul cifrei corespunzătoare afirmaţiei, litera A. Dacă apreciaţi că afirmaţia este falsă, scrieţi, pe foaia de examen, în dreptul cifrei corespunzătoare afirmaţiei, litera F şi modificaţi parţial afirmaţia pentru ca aceasta să devină adevărată. Folosiţi, în acest scop, informaţia ştiinţifică adecvată. Nu se acceptă folosirea negaţiei.

  1. Cauza sifilisului este infecţia cu HIV. F
    Cauza sifilisului este infecţia cu o bacterie.
    Notă! Această bacterie se numește Treponema Pallidum. 
  2. În timpul unei expiraţii normale, presiunea aerului din plămâni scade sub valoarea presiunii atmosferice. F
    În timpul unei inspirații normale, presiunea aerului din plămâni scade sub valoarea presiunii atmosferice.
  3. În urechea medie a mamiferelor sunt localizate ciocanul, nicovala şi scăriţa. A
