Definiție: Corp și corp comutativ

Inelul nenul se numește corp dacă orice element al acestui inel care este nenul este inversabil, adică dacă sunt îndeplinite condițiile:

  1. este inel;
  2. unde este elementul neutru al legii de compoziție iar este elementul neutru al legii de compoziție  
  3. este simetrizabil în raport cu legea de compoziție

Dacă, în plus, inelul este și comutativ, atunci corpul este corp comutativ.

Definiție: Corp finit

Spunem că un corp este corp finit dacă acesta conține un număr finit de elemente.

Exemple de corpuri:

Inelele și sunt corpuri, deoarece oricare element nenul al acestor triplete este inversabil. Aceste triplete se numesc corpuri numerice.

Inelul de matrice unde este un inel, nu este în general și un corp.

Condiția ca inelul să fie corp este ca orice matrice să fie inversabilă.
