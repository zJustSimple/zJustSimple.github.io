Număr întreg

Definiție: Număr întreg

Se numește număr întreg numărul natural 0 sau orice număr natural diferit de 0 precedat fie de semnul "+" (plus), fie de semnul "-" (minus).

Observații:

  • Mulțimea numerelor întregi se notează cu .
  • Mulțimea numerelor întregi este alcătuită din mulțimea numerelor întregi pozitive: notată și mulțimea numerelor întregi negative: .
  • Mulțimea se numește mulțimea numerelor întregi nenule (diferite de zero); se mai notează și astfel: .
  • Semnul plus, "+", din fața numerelor întregi pozitive poate fi exclus. Astfel avem mulțimea numere întregi pozitive scrisă astfel: .

Opusul unui număr întreg

Definiție: Opusul unui număr întreg

Se numește opusul unui număr întreg acel număr care se obține prin schimbarea semnului numărului considerat.

Observație:

Prin exemplele de mai sus, am observat că semnul "+" (plus) al unui număr pozitiv se schimbă la opus în semnul "-" (minus) și invers, semnul minus devine semnul plus la opus.

Reprezentarea pe axă a numerelor întregi

Numerele întregi se reprezintă pe axa numerelor astfel:

  • Pe o dreaptă d se fixează un punct O, numit origine.
  • Se alege o unitate de măsură convenabilă.
  • Se alege sensul pozitiv indicat de săgeată.

Observații:

  • Dacă parcurgem axa de la origine spre dreapta spunem că o parcurgem în sens pozitiv.
  • Dacă parcurgem axa de la origine spre stânga spunem că o parcurgem în sens negativ.

Modulul unui număr întreg

Definiție: Număr întreg mai mare ca zero

Spunem că un număr întreg b este mai mare ca 0 și notăm b>0, dacă b este un număr întreg pozitiv. 

Definiție: Număr întreg mai mic ca zero

Spunem că un număr întreg b este mai mic ca 0 și notăm b<0, dacă b este un număr întreg negativ.

Definiție: Modulul unui număr întreg

Modulul sau valoarea absolută a numărului întreg a reprezintă distanța pe axă între punctul corespunzător numărului întreg a și origine.

  • Modulul unui număr întreg pozitiv este acel număr; modulul numărului întreg 0 este 0; modulul unui număr întreg negativ este opusul acelui număr.

Ordonarea și compararea numerelor întregi

Pentru a ordona numerele întregi ținem cont de următoarele reguli:

  1. ;
  2. ;
  3. ;
  4. ;
  5. .

Observații:

  • Pentru a ordona crescător numerele întregi date într-un șir se pornește de la cel mai mic la cel mai mare.
  • Pentru a ordona numerele descrescător, se pornește de la cel mai mare număr și se merge până la cel mai mic.

Pentru a compara numerele întregi ţinem cont de următoarele reguli:

  1. Numărul întreg 0 este mai mic decât orice număr întreg pozitiv.
  2. Dintre două numere întregi pozitive este mai mare acela care are valoarea absolută mai mare.
  3. Numărul întreg 0 este mai mare decât orice număr întreg negativ.
  4. Dintre două numere întregi negative este mai mare acela care are valoarea absolută mai mică.
  5. Orice număr întreg pozitiv este mai mare decât orice număr întreg negativ.
